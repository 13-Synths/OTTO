Agregate of an in and an out point, providing operations to compare and mutate one dimensional sections.

Useful especially for marking sections on the tape

Function operator+=

Section<T>& operator+=(Section<T> o);

Add o to this section

Will, if needed, extend this section to span over both this and o

If o has size < 0, no modifications are made

Add o to a copy of this section

see <operator+=> for details

Function operator-=

Section<T>& operator-=(Section<T> o);

Subtract o from this section.

If the result cannot be expressed as a single section, no modifications are made.

  • If this contains o, no modifications are made.

  • If this is contained by or equal to o, it will be turned into a unspecified, zero-size section.

  • If the two sections partially overlap, the overlapping part will be removed.

As such, it is guarantied that every part in this, but not in o will be preserved. It is however not guarrantied that all of o will be removed.
