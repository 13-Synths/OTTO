Start the timer

Effects:

Postconditions:

Remarks:
Do not use this function. In most cases you want the free function which also pushes and pops from/to the stack

Function stop

void stop();

Stop the timer

Requires:
running == true
Effects:
pushes time_point::clock::now() - start_time to the back of data.
Postconditions:
running == false
Remarks:
Do not use this function. In most cases you want the free function which also pushes and pops from/to the stack

Function tick

void tick();

Tick the timer

Effects:
if running == false, same as start(). Otherwise, push time_point::clock::now() - start_time to the back of data, and assign time_point::clock::now() to start_time
Postconditions:
running == true
Complexity:
time_point::clock::now() is only called once.
Remarks:
Do not use this function. In most cases you want the free function which also pushes and pops from/to the stack
